About

Why

This website was created as a part of an Introduction to Digital Media class through the University of Missouri and was designed to utilize and implement fundamental digital media concepts while meeting a series of learning targets.

Digital Media Principles

Contrast

This website utilizes a two-tone color scheme I am fond of, with a light gray to provide background for different header elements on the website. The color palette was created with the online too paletton.com

Repetition

The website uses a repeating style for each page with a large header element containint the title and page number users are currently on. This is followed by a mobile style dropdown navigation menu, a title as an H2 element, and the additional information. This style is repeated across pages of the website to help consistency and viewer retention

Alignment

I designed this website to use a left aligned and mobile friendly style with ample padding between letters and elements. Content on the website is designed to be commonly aligned and consistent across multiple size windows.

Proximity

The page uses a common style sheet and spacing between different elements to keep common elements close together and commonly grouped.

Learning Outcomes

Each of the different pages on the site is designed to meet a different learning objective, promoting users toward understanding more about the fundamental components of a computer and how they interact. The first page starts with the very basics and discusses components in isolation. This is meant to give users an idea of the function of each device, abstracting away how they interact for the time being. After viewieng this, users understand what the components are and that we use 1's a nd 0's, but not why. The audio component answers this question which is important to understand before proceeding further. After listenting to this audio, viewers should understand more about why we design computers to use 1's and 0's. The next step is to put this information together and learn about how these different components and bits interact to form, on an abstract level, the devices we use today. This is accomplished with a video illustrating each component and how they interact. After viewing this an essay style quiz is available for users to take to demonstrate their understanding. Each of the questions on this quiz is derived directly from the learning objects for each page itself.
